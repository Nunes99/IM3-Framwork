# IM³ Tilda Dashboard — GitHub Pages Package

Files:
- index.html: full dashboard structure with external CSS and JS.
- im3-style.css: visual styling.
- im3-app.js: application logic.
- tilda-embed.html: minimal iframe snippet to paste in Tilda.

Deploy:
1. Create a GitHub repository.
2. Upload these files to the repository root.
3. Enable GitHub Pages from Settings > Pages > Deploy from branch > main > /root.
4. Copy the GitHub Pages URL.
5. Replace YOUR_USERNAME and YOUR_REPOSITORY inside tilda-embed.html.
6. Paste tilda-embed.html into a Tilda HTML block.
